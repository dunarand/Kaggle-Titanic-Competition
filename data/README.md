# Data

The raw data used in this project is obtained from the official competition
webpage at https://www.kaggle.com/competitions/titanic/data which you can
obtain by using the Kaggle API:

```bash
kaggle competitions download -c titanic
```

Unzip the contents to the `raw` subdirectory.

Throughout the project, this raw data is going to be modified and saved to the
`modified` subdirectory. Make sure that it is present in the project directory.
